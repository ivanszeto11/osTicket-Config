# osTicket - Admin Panel Configurations

## Overview

This project demonstrates the post-installation configuration of an open-source help desk ticketing system, osTicket. We will explore settings in the admin panel, including Roles, Departments, Teams, Agents, SLAs, and Help Topics. Upon completion of the lab, we will have a better understanding of the importance of these configurations in the functionality of osTicket and how to modify them.  

### Environments and Technologies Used

- Microsoft Azure (Virtual Machines)
- Remote Desktop Connection
- Windows 11 Pro
- osTicket
